# 🎯 Features Overview

## 1. AI Toxicity Detection

**What it does:**
- Analyzes content for toxic language
- Provides toxicity scores (0-1 scale)
- Explains why content is flagged
- Offers confidence percentages

**Tech:**
- OpenAI API + Perspective API
- ML models for pattern recognition
- Real-time analysis

**Demo:** Visit Dashboard → Live Feed
